# Real-time Interactive Kinematic Motion Generation

## Features

* **Data-Driven**: Leverages the [LAFAN1 Retargeting Dataset](https://huggingface.co/datasets/lvhaidong/LAFAN1_Retargeting_Dataset) for training.
* **Model Architecture**: A **light-weight**, **purely kinematics**, **auto-regressive** generative model.
* **Real-Time Inference**: Capable of processing user inputs instantly to generate responsive motion in real-time.
* **Interactive Visualization**: Integrated with **Rerun SDK** for real-time 3D visualization of the robot's kinematic chain and trajectory.
